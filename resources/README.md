# Resources

Curated external links for SolidWorks learning, example files, community CAD, and API automation. These are not part of the MK-Brace project; they are reference material for contributors and future design work.

## Training & exercise repositories

| Resource | Link | Notes |
| --- | --- | --- |
| *SOLIDWORKS 2016 Essential Training* (Tertiary Infotech) — exercise files | [tertiarycourses/SolidworksTraining](https://github.com/tertiarycourses/SolidworksTraining) | Official course materials for a structured essentials curriculum; [course outline](https://www.tertiarycourses.com.sg/solidworks-2016-essential-training.html) is listed in that repo. |
| *Learn SOLIDWORKS* (second edition, Packt) — chapter project files | [PacktPublishing/Learn-SOLIDWORKS-Second-Edition](https://github.com/PacktPublishing/Learn-SOLIDWORKS-Second-Edition) | Book companion assets organized by chapter; see the book README for the intended software and chapter mapping. |

## SolidWorks + Python (API automation)

| Resource | Link | Notes |
| --- | --- | --- |
| SW ToolKit | [Glutenberg/swtoolkit](https://github.com/Glutenberg/swtoolkit) | Python toolkit to script SolidWorks (batch exports, build automation, etc.). **Requires** a local SolidWorks install and a Windows environment suitable for the COM API. |

## Community CAD

| Resource | Link | Notes |
| --- | --- | --- |
| GrabCAD | [grabcad.com](https://grabcad.com/) | Public library of user-uploaded models; useful for reference geometry, fastener/standard parts, and design inspiration. Check each model’s **license and terms of use** before reusing. |
